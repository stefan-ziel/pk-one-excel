# pk-one-excel
* Kompiliert und getestet mit JACOB Version 1.14.3 das passende dll bitte im java.library.path ablegen.

* Pfad zum Excel wird per JNI ermittelt. Kontext des Servlets wie unter conf

* Makros müssen in der Konfiguration von Excel freigeschaltet sein

* Abbildung von JSON-Variablen auf Zellen über die Konstanten im Kopfbereich des Servlet.
